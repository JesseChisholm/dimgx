**TL;DR**
---------

``dimgx`` extracts and flattens `Docker <https://www.docker.com/whatisdocker/>`_ `image <https://docs.docker.com/terms/image/>`__ `layers <https://docs.docker.com/terms/layer/>`__. It is licensed under the `MIT License <http://opensource.org/licenses/MIT>`_.

See `the docs <https://dimgx.readthedocs.org/>`__ for more information.
